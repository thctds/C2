PAra a compilação do programa, apenas é necessário executar o comando "$ make all" na raiz do programa. Ao ser executado, será gerado o documento de Log, que é atualizado cada vez que o programa é utilizado.
Uma vez compilado, o programa pode ser rodado a partir do comando "lp2-Thiago".

O programa é capaz de identificar imagens em formato Jpg, documentos HTML e arquivos de áudio em mp3. Amostras, como requerido na especificação, contidos na pasta files.
